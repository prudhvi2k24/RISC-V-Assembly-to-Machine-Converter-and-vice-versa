    Add registers

    sub rd, rs1, rs2
    Subtract registers
  • SLT
    slt rd, rs1, rs2
    Set if less than (signed)
  • SLTI
    slti rd, rs1, imm
    Set if less than immediate (signed)
  • SLTU
    sltu rd, rs1, rs2
    Set if less than (unsigned)
  • SLTIU
    sltiu rd, rs1, imm
    Set if less than immediate (unsigned)

About RISC-V

RISC-V is an open-source instruction set architecture (ISA) that is revolutionizing the world of computer architecture. It provides a foundation for building efficient, customizable processors for various applications.

About This Tool

This converter helps you understand and work with RISC-V assembly language by:

  • Converting RISC-V assembly instructions to their corresponding machine code
  • Translating machine code back to human-readable assembly instructions
  • Supporting various instruction formats (R-type, I-type, S-type, etc.)
  • Providing detailed breakdowns of instruction fields and their binary representations

Whether you're learning computer architecture, debugging assembly code, or developing RISC-V software, this tool can help you understand the relationship between assembly language and machine code.
